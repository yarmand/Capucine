    
    Usage:

      Configure your preferences in capucine.yaml

      capucine [option]
  

    Options:

      new (or n) [name]   -   Create a new project (if not name, create the folder : capucine )

      init (or i)         -   Initialize a project in current folder (simply add a capucine.yaml file)
      
      compile (or c)      -   Compile .sass, .coffee, .erb files (just once)
                              
      watch (or w)        -   Compile .sass, .coffee, .erb files (run the watcher).

      help                -   Print this message
